## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/GourangaDasSamrat/Gradient-Generator.git
   ```
2. Navigate to project directory:
   ```bash
   cd Gradient-Generator
   ```
3. Open `index.html` in your browser
